img로 이미지 넣기

<img src="이미지경로" alt="대체텍스트">
이미지경로 - 이미지 정보
대체텍스트 - 음성낭독기용, 이미지 나타나지 않을 때 대비용 설명문.

  • 절대경로 - 항상 고정된 주소.(주로 외부 사이트용)
  • 상대경로 - 웹문서의 위치에 따라 달라지는 주소.
    (상위로 이동시는 ../ 를 횟수만큼, 하위로 이동시는 폴더명/ 로)
    [ex] "../img/abc.jpg" - 상위폴더의 또다른 하위폴더인 img폴더 내에 abc.jpg파일이 있음

웹에서 사용하는 대표 이미지형식 3가지

  • *.gif - 256가지 색상만 표현. 투명배경 지원. 움짤 가능.
  • *.jpg - 트루컬러 색상 표현 가능. 실사사진에 적합.
  • *.png - 투명배경 지원. 트루컬러 색상 표현 가능.

width/height="n" 가로/세로 크기 지정

  2. width="400"
  3. height="400"
  4. width="640" height="426"
    (원본크기 그대로 적용시에도 가로세로 크기를 기입하면 브라우저에 이미지가 빨리 뜸.)
