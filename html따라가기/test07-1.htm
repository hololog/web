a로 내부링크 걸기

페이지가 길어지면 해당 페이지내에서 위나 아래로 이동하는 링크를 걸 수 있다.
이동하고자 하는 곳의 tag(태그)에 id를 추가하고 (id값은 고유한 값이므로 주소로 씀) 링크를 걸 때 href의 값에 해당 id를 주소로 활용.(#id) 

  <a href="#abc"> ... </a> = id가 abc인 곳을 찾아 이동(내려감)
          ... 
  <tag id="def">
          ...
          ...
  <tag id="abc">
          ...  
  <a href="#def"> ... </a> = id가 def인 곳을 찾아 이동(올라감)
