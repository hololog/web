html?

Hyper Text Markup Language의 약자.
웹페이지를 만드는 기본 프로그램언어.
구조를 담당.

웹페이지의 필수(기본)tag들

 html          -----  웹페이지의 시작을 선언
  head          -----  머리말의 시작을 선언
   title ... /title   ----- 제목(탭영역의) 지정  
  /head         -----  머리말의 마무리를 선언
  body          -----  본문의 시작을 선언
  /body         -----  본문의 마무리를 선언
 /html         -----  웹페이지의 마무리를 선언

body안의 기초 tag들

p = 문단 태그
br = 줄바꿈 태그
hr = 수평선 태그
hn = 제목글자 태그 (n=1~6)
pre = 소스상의 공백을 그대로 살려주는 태그 (고정폭글꼴)
