[ fomr-2 ] html5에서 추가된 input type값들

  1. input type="time" 시간(오전/오후 시/분)을 지정하는 요소
  2. input type="date" 날짜(연월일)을 지정하는 요소
    3. (min="1900-01-01" max="2050-12-31")
  3. input type="datetime-local" 날짜(연월일)와 시간을 지정하는 요소
  4. input type="month" 날짜(연월)를 지정하는 요소
  5. input type="week" 날짜(연주)를 지정하는 요소
  6. input type="number"
    • (min="0")-최소값 지정시 min속성 추가
    • (max="8")-최대값 지정시 max속성 추가
    • -최대, 최소 동시사용
    • (step="30")-증감 간격 지정시 step속성 추가
  7. input type="range" 범위내의 값을 지정하는 슬라이드바
    모바일 브라우저에서 가상 키패드가 숫자로 바뀌고 전화걸기 가능.
  8. input type="tel" 전화번호 값을 지정하는요소/모바일 환경에서
  9. input type="url" 웹주소값을 지정하는 요소
    모바일 브라우저에서 가상키패드가 http://, com등이 나올수 있고
    웹주소형식의 값이 바른지 브라우저에서 체크함(@유무, @이후값 유무)
  10. input type="email" 이메일 주소 값을 지정하는 요소
    모바일 브라우저에서 가상키패드가 @등이 나올수 있고
    email형식의 값이 바른지 브라우저에서 체크함(@유무, @이후값 유무)
  11. input type="color" 색상값을 지정하는 요소

html5에서 추가된 input속성들

  1. autofocus - 자동 포커싱. 해당욧가 입력대시상태로
  2. autocomplete="on/off" - 자동완성기능을 켜거나 끔
  3. placeholder - " "안에 입력예시나 설명문 등을 지시
  4. required - 필수입력란
  5. readonly - 읽기만 가능한 입력 요소지정(입력값전송o)
  6. disabled - 사용불가 요소 지정(입력값전송x)
maxlength="n" - 실제 입력가능한 데이터 길이 size="n" - 화면상의 상자 길이 지정
