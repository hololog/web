[ form-2 ] html5에서 추가된 input type값들

  1. input type="time" 시간(오전/오후 시분)을 지정하는 요소
  2. input type="date" 날짜(연월일)를 지정하는 요소
    (min="1900-01-01" max="2250-12-31")
  3. input type="datetime-local" 날짜와 시간(연월일 오전/오후 시분)을 지정하는 요소
  4. input type="week" 날짜(연주)를 지정하는 요소
  5. input type="month" 날짜(연월)를 지정하는 요소
  6. input type="number" 숫자값을 지정하는 요소
    (min="0") - 최소값 지정시 min속성 추가
    (max="5") 최대값 지정시 max속성 추가
    (step="30") 증감 간격 지정시 step속성 추가
  7. input type="range" 범위내의 값을 지정하는 요소
  8. input type="tel" 연락처(전화번호)값을 지정하는 요소.
    모바일브라우저에서 가상키패드가 숫자로 바뀌고 전화걸기 가능.
  9. input type="email" 연락처(이메일주소)값을 지정하는 요소
    모바일브라우저에서 가상키패드가 @ 등이 나올 수 있고
    email형식의 값이 바른지 브라우저에서 체크함(@유무, @이후값 유무)
  10. input type="url" 연락처(웹주소)값을 지정하는 요소
    모바일브라우저에서 가상키패드가 http://, .com 등이 나올 수 있고
    웹주소형식의 값이 바른지 브라우저에서 체크함(프로토콜 유무)
  11. input type="color" 색상값을 지정하는 요소

html5에서 추가된 input속성들

  1. autofocus - 자동포커싱. 해당 요소가 입력대기 상태로
  2. autocomplete="on/off" - 자동완성 기능을 켜거나 끔.
  3. placeholder="***" - 입력예시나 설명문 등을 지시
  4. required - 필수입력 요소로 지정
  5. readonly - 읽기만 가능한 입력 요소로 지정 (입력값 전송)
  6. disabled - 사용불가한 요소로 지정 (입력값 전송 불가)
maxlength="n" - 실제 입력가능한 데이터 길이 지정(최대값), size="n" - 화면상의 상자 길이 지정.

fieldset, legend로 form요소들을 그룹화하여 나타내기

비슷한 유형의 폼요소들이나 특정 주제의 서로 연관된 폼요소들끼리 그룹화할 때
fieldset태그로 묶어주고 해당 그룹의 제목은 legend태그로 지정.
