기타 form요소들

1. textarea ... /textarea 주관식. 여러줄 큰 글상자

시작과 종료의 한 쌍.
가로는 cols속성, 세로는 rows속성으로 지정

2. select... /select 공간절약형 객관식. 드롭다운 리스트(펼침목록)

select로 전체를 묶어주고 실제 선택항목은 option로 지정.

출근시 소요 시간은?

select에 size="n"속성으로 n개의 항목이 보이는 상자형으로 지정.

팀회의 희망 시간은?

option에 selected(selected="selected")속성으로 option들 중에서 초기값으로 지정.

팀회의 희망 시간은?

희망 상담시간은?

select에 multiple(multiple="multiple")속성으로 다중선택 가능형으로 지정.

상자형으로 바뀌며 2개 이상 선택시 ctrl키나 shift를 누름. (size="n"속성도 동시 사용 가능)

팀회의 희망 시간은?

option들을 optgroup으로 그룹화하기

여러 연관있는 option들을 optgroup으로 묶어서 그룹화하여 표현.
해당 optgroup의 제목(주제)는 label속성으로 지정.

3. button ... /button - 전송버튼(취소/일반)

  • - 버튼내에 글자관련 태그 적용 가능
  • - type="reset"로 취소용 버튼으로 변경
  • - type="button"로 (폼과 무관한)일반 버튼으로 변경
  • - img src로 버튼내에 이미지 넣기. (+ css로 버튼내에 웹아이콘을 추가 가능)

9 . input type="submit/reset/button" - 정보를 전송/초기화/(일반) 하는 버튼
