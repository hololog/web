background - 배경

  1. background-color: ; 배경 색상
  2. background-image:url("이미지경로"); 배경 이미지

background-image의 속성들

  • background-repeat: ; 반복 지정
    1. repeat 기본값. 양사방 반복
    2. repeat-x 가로로만 반복
    3. repeat-y 세로로만 반복
    4. no-repeat 반복 안함 (한번만)
  • background-size: ; 크기 지정
    1. Xn Yn 가로 세로 순으로 직접 지정
    2. cover 영역을 이미지 하나로 다덮기 (이미지가 잘려나갈 수도 있음)
    3. contain 이미지 하나로 영역을 가로나 세로에 최대한 맞춰서 덮기 (이미지가 잘려나가지 않음)
  • background-position: ; 위치 지정
    1. Xn Yn 가로 세로 순으로 직접 지정
    2. left/center/right top/center/bottom 방향값을 조합하여 지정
  • background-attachment: ; 고정여부 지정
    1. scroll
    2. fixed
    3. local

background-image 2개 적용하기

background-image:url("이미지1경로"), url("이미지2경로");
배경 이미지의 경로를 ,로 두번 등록하고 나머지 속성들도 ,로 각각 적용 가능.

background-attachment: ; 고정여부 지정

scroll - 자신의 영역 스크롤바에는 고정/브라우저 스크롤바에는 스크롤. fixed - 자신의 영역 스크롤바와 브라우저 스크롤바에 모두 고정. local - 자신의 영역 스크롤바와 브라우저 스크롤바에 모두 스크롤.
