margin 요소의 외부여백

margin의 각 방향별 지정 세부속성

  • margin-top:n;
  • margin-right:n;
  • margin-bottom:n;
  • margin-left:n;

margin의 값 갯수별 지정법

  • margin:n; 양사방 동시에 같은 값을 적용
  • margin:n n ; 상하, 좌우가 서로 같은 값
  • margin:n n n ; 상, 좌우가 서로 같은 값, 하의 순으로 적용
  • margin:n n n n; 상, 우, 하, 좌 의 순으로 각각 적용
