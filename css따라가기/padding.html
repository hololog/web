padding 요소의 내부여백

padding의 각 방향별 지정 세부속성

  • padding-top:n;
  • padding-right:n;
  • padding-bottom:n;
  • padding-left:n;

padding의 값 갯수별 지정법

  • padding:n; 양사방 동시에 같은 값을 적용
  • padding:n n ; 상하, 좌우가 서로 같은 값
  • padding:n n n ; 상, 좌우가 서로 같은 값, 하의 순으로 적용
  • padding:n n n n; 상, 우, 하, 좌 의 순으로 각각 적용
