display 요소가 화면에 나타나는 방식(성격) 지정

  • display:inline; block인 요소를 inline으로 변경
  • display:block; inline인 요소를 block으로 변경
  • display:inline-block; 어떤 요소든지 inline과 block의 특징을 결합한 방식으로 변경
  • display:none; 요소가 화면에서 사라짐

inline, block의 특징

  1. inline - 한 줄에 나란히 옴. (자동으로 줄 안바뀜)
    크기 적용 불가, margin은 위아래 적용 불가.
  2. block - 다른 요소와 나란히 올 수 없음.
    크기 적용, margin 모두 적용.

display:inline; block요소를 inline으로 변경

[ block ] aside tag
[ block ] div tag

display:block; inline요소를 block으로 변경

[inline] a tag [inline] b tag [inline] span tag
